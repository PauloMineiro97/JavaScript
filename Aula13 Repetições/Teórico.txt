As estruturas de repetições usam While e coloca um bloco dentro do {}

While(condição) {

}


Criar uma variável 
e enquanto o valor da variável for menor ou igual a 6, será repetida a mensagem 'Tudo bem'?
c++ = variável c+1, c+2, c+3 ...

var c = 1
while(c<=6) {
    console.log('Tudo bem?')
    c++
}
